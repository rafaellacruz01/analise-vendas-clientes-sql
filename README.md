# 📊 Análise de Vendas e Clientes com SQL

## 📌 Sobre o projeto
Este projeto tem como objetivo analisar dados de vendas e comportamento de clientes utilizando SQL, com foco no uso de INNER JOIN para cruzamento de informações.

---

## 🎯 Objetivo
Identificar padrões de compra, clientes mais relevantes e regiões com maior volume de vendas.

---

## 🛠️ Ferramentas utilizadas
- SQL
- Excel
- GitHub

---

## 📊 Análises realizadas
- Total de vendas por cliente  
- Vendas por cidade  
- Quantidade de compras por cliente  
- Vendas por dia da semana  

---

## 💡 Insights de negócio
- Identificação de clientes com maior valor de compra  
- Regiões com maior volume de vendas  
- Padrões de consumo ao longo da semana  

---

## 📂 Estrutura do projeto
- data → base de dados  
- sql → consultas SQL  
- dashboard → visualizações
